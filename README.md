## 📈 Dashboard Features

### 🔹 Monthly Quantity Ordered
- Analyzes monthly order quantity trends
- Identifies peak sales months

### 🔹 Product Code Analysis
- Shows count of entries by product code
- Helps identify popular products

### 🔹 Order Status Distribution
- Displays shipped, cancelled, disputed orders
- Easy business performance tracking

### 🔹 Vehicle Wise Analysis
- Compares quantity ordered by vehicle category
- Includes:
  - Classic Cars
  - Vintage Cars
  - Motorcycles
  - Trucks
  - Ships
  - Planes
  - Trains
